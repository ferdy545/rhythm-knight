# Rhythm Knight

Rhythm Knight é um jogo de plataforma com mecânicas de jogos de ritmo, desenvolvido durante o primeiro semestre de 2025 no programa de treinamento de desenvolvimento de jogos do grupo [USPGameDev](usp.game.dev.br).

Nota: Rhythm Knight foi desenvolvido colaborativamente no GitLab. Este repositório é apenas uma réplica (_mirror_) do repositório original e não será modificado. Você pode acessar o repositório original do Rhythm Knight, no GitLab [aqui](https://gitlab.com/uspgamedev/treinamento/2025-1-platformer-de-ritmo).
