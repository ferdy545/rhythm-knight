# Rhythm Knight

Rhythm Knight é um jogo de plataforma com mecânicas de jogos de ritmo, desenvolvido durante o primeiro semestre de 2025 no programa de treinamento para desenvolvimento de jogos do grupo [USPGameDev](usp.game.dev.br).

Este jogo foi feito com o motor de jogo [Godot](godotengine.org/).

## Informações Importantes

Rhythm Knight foi desenvolvido colaborativamente no _GitLab_. Este repositório é apenas uma réplica (_mirror_) do repositório original. Você pode acessar o repositório original do Rhythm Knight, no _GitLab_ [aqui](https://gitlab.com/uspgamedev/treinamento/2025-1-platformer-de-ritmo). _Data de execução do mirror: 02 de maio de 2026._

O processo de desenvolvimento deste jogo está inativo, e atualmente este repositório está servindo apenas para preservá-lo.

## _Gameplay_

Atualmente há apenas um nível (fase). O objetivo é chegar até o final dela. Para isso, o jogador deve evitar perigos na fase, o que inclui obstáculos e inimigos. 

Para iniciar um combate com um inimigo, o jogador deve aproximar-se o suficiente dele, aguardar a exibição da sequência de teclas que devem ser pressionadas para derrotá-lo e por fim repetir com sucesso essa sequência no mesmo ritmo demonstrado.

O jogador não pode sair do combate, a não ser que ele elimine o inimigo ou seja eliminado por ele.

### Controles (teclas)

- A: movimentação à esquerda <br>
- D: movimentação à direita <br>
- W: pulo <br>
- Direcionais (↑, ↓, ←, →): ataques ("_parry_") durante o combate
